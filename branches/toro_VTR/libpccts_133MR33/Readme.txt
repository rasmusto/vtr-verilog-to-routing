This directory contains the Purdue Compiler Construction Tool Set C++
library code for Toro (Windows version).

To build on Windows VisualC++...

   1. run VisualC++ (2010 or 2012)
   2. open "h/libpccts_h.vcxproj"
   3. build
   4. open "pcctsGeneric/libpccts_generic.vcxproj"
   5. build
